关于注册表

可以使用 Internet 信息服务管理单元配置多种 Internet 服务。在注册表中，不是用 Internet 信息服务管理单元配置的配置参数要么特定于某一服务，要么是针对 IIS 和所有服务的全局参数。

应尽可能使用 Internet 信息服务管理单元更改 Internet 服务器设置。要使对注册表的更改生效，必须重新启动此次更改所影响的服务。对于全局项目，则必须重新启动所有服务。

要点   不正确地使用“注册表编辑器”可能会导致某些问题，包括网站或 FTP 站点失败。如果使用时出错，可能会损坏网站或 FTP 站点的配置。应该只为在用户界面中无法调整的设置编辑注册表项，并且在直接编辑注册表时一定要小心。

IIS 的早期版本中存在的许多注册表项已经移植到一个称为配置数据库的、可快速访问的、具有分层结构的数据库中。保留的注册表项用于启动 Internet 服务，并保证与 IIS 早期版本的向后兼容性。新写入的所有信息都将写入配置数据库。详细信息，请参阅配置数据库。

注册表属性页项

注册表的属性以表的形式出现。配置信息的组织形式如下：

标识符： 表示注册表属性的名称 (DWORD)。
数据类型： 标识数据的变量类型的 DWORD。
范围： 属性的取值范围。
默认值： 用户未更改时，属性的值。

使用内容分级

可以配置 Web 服务器的内容分级功能，将说明性标签嵌入网页的 HTTP 头中。某些 Web 浏览器（例如 Microsoft Internet Explorer 3.0 或更高版本）可以检测到这些内容标签，并帮助用户识别那些不受欢迎的 Web 内容。Web 服务器默认的基于分级系统的 Internet 内容选择平台 (PICS) 使用 Recreational Software Advisory Council (RSAC) 开发的系统，此系统根据暴力、裸体、性及冒犯性语言的级别对内容进行分级。设置 Web 内容分级之前，应填写 RSAC 问卷了解为您的特定 Web 内容建议的内容分级。

对内容进行分级
  1. 在 Internet 信息服务管理单元中，选择某个网站、目录或文件，并打开其属性页。
  2. 单击“HTTP 头”选项卡，然后单击“内容分级”下的“编辑分级”。
  3. 在“分级服务”属性页中的“RSAC 注册”下，单击“分级问卷”，然后按照 RSAC 提供的指导进行操作。
  4. 单击“确定”。

 

设置内容分级
  1. 在 Internet 信息服务管理单元中，选择某个网站、目录或文件，并打开其属性页。
  2. 单击“HTTP 头”选项卡，然后单击“内容分级”下的“编辑分级”。
  3. 在“分级”属性页中，选中“对资源启用分级”复选框。
  4. 在“类别”文本框中，单击分级类别。使用分级标尺设置不受欢迎内容所属类别的级别。每项设置还将显示对指定等级的说明。
  5. 在“内容分级人员的电子邮件”中，输入对内容进行分级的人员的电子邮件地址，然后选择“过期日期”列表，并从日历中选择分级的过期日期。
  6. 单击“确定”。

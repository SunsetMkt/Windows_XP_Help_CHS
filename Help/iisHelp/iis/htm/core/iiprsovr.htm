设置 FTP 消息和目录输出样式

建立 FTP 站点时，可以向用户发送消息提供有关此站点的详细信息。可以采用欢迎消息（当用户第一次登录时欢迎用户访问 FTP 站点）和退出消息（当用户注销时显示）的形式。也可以设计一个消息提示用户已经达到了最大连接数（可以在“FTP 站点”属性页中设置 FTP 连接限制)。

设置 FTP 欢迎、退出或最大连接数消息
  1. 在 Internet 信息服务管理单元中，选择要设置消息的 FTP 站点。
  2. 用鼠标右键单击站点并选择“属性”。
  3. 选择“消息”属性页。
  4. 在相应文本框中分别输入欢迎、退出或最大连接数消息。

也可以设置 FTP 站点的目录输出样式。目录输出样式定义了 FTP 客户端发送 LIST 操作时所使用的格式；目录输出样式可以是 MS-DOS，也可以是 UNIX。默认目录输出样式是 MS-DOS。

默认情况下，MS-DOS 目录列表样式以两位数字格式显示年份。可以通过启用 FtpDirBrowseShowLongDate 配置数据库属性，将其更改为使用四位数字显示年份。

当文件日期与 FTP 服务器的年份不一致时，UNIX 目录列表样式以四位数字显示年份。如果文件日期与 FTP 服务器年份相同，将不返回年份。

设置 FTP 站点的目录输出样式
  1. 在 Internet 信息服务管理单元中，选择要设置目录输出样式的 FTP 站点。
  2. 用鼠标右键单击此站点并选择“属性”。
  3. 选择“主目录”属性页。
  4. 在“目录列表样式”下，选择 UNIX 或 MS-DOS。

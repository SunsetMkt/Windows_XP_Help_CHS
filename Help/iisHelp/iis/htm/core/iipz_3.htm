“主目录”属性页

可以将主目录的位置更改为下列位置：

  • 此计算机上的目录。
  • 另一台计算机上的共享。

出现提示时，请输入访问计算机所需的用户名和密码。

FTP 站点目录

在文本框中，输入目录或目标 URL 的路径。语法必须与所选路径类型相匹配：

对于本地目录，应使用完整路径，例如，C:\Catalog\Shoes。也可以不输入路径，而单击“浏览”按钮选择本地目录。

对于网络共享，应使用通用命名约定 (UNC) 服务器和共享名，例如，\\Webserver\Htmlfiles。

如果要选择本地目录，但不想在“本地路径”文本框中输入目录路径，请单击“浏览”。

读取

选择此选项，允许用户阅读或下载存储在主目录或虚拟目录中的文件。

写入

如果允许用户将文件上载到服务器上已启用的目录中，请选中此选项。仅对准备接收用户文件的目录选择此权限。

记录访问

如果要将对此目录的访问记录到日志文件，请选中此复选框。只当此 FTP 站点启用了日志记录时方可记录访问。默认情况下，启用日志记录。

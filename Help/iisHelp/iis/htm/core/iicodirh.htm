更改主目录

每个网站都必须有一个主目录。如果在同一台计算机上同时设置了网站和 FTP 站点，必须为每种服务（WWW 和 FTP）指定不同的主目录。WWW 服务的默认主目录是 C:\Inetpub\Wwwroot。FTP 服务的默认主目录是 C:\Inetpub\Ftproot，也可以选择其他目录作为主目录。

更改主目录
  1. 在 Internet 信息服务管理单元中，选择某个网站或 FTP 站点，并打开其属性页。
  2. 单击“主目录”选项卡，然后指定主目录的位置。可以选择：
    • 此计算机上的目录
    • 另一台计算机上的共享
    • 重定向到 URL

      • 注意   请求此 URL 的浏览器将转到新的 URL。无法重定向 FTP 目录。

  3. 在文本框中，指定目录的路径名、共享名或 URL。

    4. 注意   如果选择了网络中的某个共享目录，可能需要输入用户名和密码才能访问此资源。建议使用 IUSR_computername 帐户。如果使用具有服务器管理权限的帐户，客户端将具有了对服务器进行操作的权限，这将严重威胁网络的安全性。

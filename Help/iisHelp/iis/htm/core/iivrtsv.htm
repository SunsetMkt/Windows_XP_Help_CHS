网站和 FTP 站点

起初，每个域名（如 www.microsoft.com）只表示一台计算机。而现在，运行 Windows 2000 Server 的一台计算机可以同时托管多个网站或 FTP 站点，看上去好像有几台计算机。每个网站都具有托管一个或多个域名的能力。由于每个站点都模仿一台计算机，因此有时也将站点称为“虚拟服务器”。

本节包含：

  • 关于网站和 FTP 站点：描述网站和 FTP 站点及其属性的概念信息。
  • 启动和停止站点：有关使用 Internet 信息服务管理单元启动和停止站点的过程信息。
  • 设置 FTP 消息和目录输出样式：为 FTP 客户端创建自定义消息和目录列表格式。
  • 命名网站：将唯一的主机头名、IP 地址和端口组合分配给网站。
  • 重新启动 IIS：通过 Internet 信息服务管理单元停止和启动 Internet 服务。
  • 更改继承的默认值：更改站点的全局默认设置。
  • 备份配置数据库：备份 IIS 配置设置。
  • 还原配置数据库：还原 IIS 配置设置。

本节不包含：

  • 网络命名系统；请参阅名称解析。
  • 增强安全性设置；请参阅安全性。
  • 添加虚拟目录；请参阅创建虚拟目录。

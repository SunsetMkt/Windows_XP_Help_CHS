使用文本编辑器创建网页

可以使用记事本等标准文本编辑器，通过输入 HTML 标记来创建网页。HTML 标记是一些格式化指令，它告诉 Web 浏览器如何显示文档。标记由一对包含在小于号和大于号之间的文字组成 (< >)。例如，下列 HTML 代码将 Hello World 用粗体显示在 Web 浏览器上：<B>Hello World</B>。显示为 Hello World。

也可以嵌套多个标记以应用多种属性。例如，如果只对 World 应用斜体，而对整句话应用粗体，则应采用如下形式：<B>Hello <I>World</I></B>。结果显示为：Hello World。注意应该避免标记重叠；下例是一个无效的 HTML 语法，并且根据 Web 浏览器的不同可能会产生不正常的结果：<B>Hello <I>World</B></I>。

使用文本编辑器创建网页
  1. 在标准文本 (ASCII) 编辑器中创建一个基本框架或模板，为浏览器提供一些重要的 HTML 标题和格式化信息。输入下列标记：
    2. <HTML>
<HEAD>
  <TITLE>  </TITLE>
</HEAD>
  <BODY>

  </BODY>
</HTML>
  2. 将网页的标题放在 <TITLE> 标记之间。标题显示在 Web 浏览器的标题栏上而不是显示在页面中。选择标题时，请尽量简要地说明网页的内容。
  3. 将要在 Web 浏览器上显示的正文和其他内容放在 <BODY> 标记之间。
  4. 将 HTML 文件用 .htm 扩展名保存（确保文本编辑器不在文档后自动添加 .txt 扩展名），然后在 Web 浏览器中打开此文件并查看所做的更改。
  5. 将其他元素或格式化信息添加到网页中；有关信息，请参阅下列 HTML 标记表。重复第 4 步以查看所做的更改。

提示   应阅读 HTML 规范（Internet 上的许多网站都有提供）以充分利用并理解 HTML。另外，可以使用 Web 浏览器的源文件查看功能可以查看几乎所有网页的 HTML 源文件。您可以使用此功能学习其他 HTML 编写者是如何设计他们的网页的。

 

HTML 标记 说明
<P> </P> 段落的起始和结束。
<Hn> </Hn> 标题文本的起始和结束。值 n 表示不同的字体大小，应该是 1 到 6 的整数。
<U> </U> 下划线文本的起始和结束。
<UL> </UL> 项目符号列表的起始和结束。每一条列表项必须以 <LI> 标记开头。
<OL> </OL> 项目编号列表的起始和结束。每一条列表项必须以 <LI> 标记开头。
<LI> 表示一个项目列表。不需要结束标记。
<B> </B> 粗体的起始和结束。
< IMG SRC=image file> 将图形文件插入网页。
< BGSOUND SRC=sound file> 加载页面后播放背景音乐。注意   此特性只有 Microsoft Internet Explorer 4.0 或更高版本才支持。
<FONT SIZE=n FACE=font type> < /FONT > 文本字体特征（如大小、颜色或字体）的起始和结束。
<A HREF=file > </A> 创建到文件中某个位置或其他文档的超级链接。例如，下面将创建一个链接，用户单击它可以查看另一个 HTML 文档：<a href="mainpage.html"> 返回主页</a>
<HTML> </HTML> 表示文件内容是以 HTML 语言编写的。通常，它们是 HTML 文档的第一个和最后一个标记。
<HEAD> </HEAD> 包含 HTML 文档标题信息，例如文档标题和其他描述性的信息。Web 浏览器和网页搜索程序可以使用这些信息。但是不应将文档正文放在此标记之内。
<TITLE> </TITLE> 表示 HTML 文档的标题。标题显示在 Web 浏览器的标题栏中，而不在正文中显示。
<BODY> </BODY> 包含要在网页中显示的内容。

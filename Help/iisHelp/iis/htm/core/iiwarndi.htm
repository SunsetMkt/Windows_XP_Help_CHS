终止独立的应用程序

无须终止 Web 服务器即可以终止应用程序并将其从内存中卸载。这使得应用程序的开发和调试变得更为容易。

终止独立的应用程序
  1. 在 Internet 信息服务管理单元中，选择网站或应用程序的开始位置目录。
  2. 打开目录的属性页，然后单击“主目录”、“虚拟目录”或“目录”选项卡。

    3. 如果您所在的目录列为“开始位置”目录，则“应用程序名”框已被填写。

  3. 单击“卸载”按钮。

    4. 如果“卸载”按钮无效，则表明当前目录不是应用程序的开始位置目录。

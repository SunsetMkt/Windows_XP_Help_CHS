“服务器端包含”参考

可以使用“服务器端包含 (SSI)”指令在静态网页中执行某些任务。静态网页在执行大规模任务时比 ASP 网页快一些。使用 SSI 指令，可以在网页中包含文件的内容、与文件有关的信息、服务器变量或可执行文件（如 CGI 文件和 ISAPI 应用程序）的输出。

除了包含可执行文件之外，所有这些任务都可以通过 ASP 脚本执行。如果需要将 ASP 脚本用于其他网页任务，则需要使用这些任务的 ASP 解决方案。除了 #include 指令外，不能将其他 SSI 指令与 ASP 脚本混合使用。

注意   服务器端时间和日期函数使用由 C 库区域设置（而不是操作系统区域设置）设定的语言。C 库区域设置通常设为英语。

在本节中：

  • #config：指定用于 SSI 错误信息的字符串，以及由 #flastmod 和 #fsize 返回的日期和文件大小的格式。
  • #echo：将服务器变量的值插入 HTML 网页。
  • #exec：运行应用程序或 Shell 命令，并将输出结果插入到 HTML 网页中。
  • #flastmod：将文件修改日期插入网页，其格式由 #config 指令指定。
  • #fsize：将文件大小插入网页，其格式由 #config 指令指定。
  • #include：将文件包含在网页中。这是唯一一个既可用于 ASP 文件又可用于 STM 文件的指令。

使用 IIS 快速建立站点

安装 Windows XP Professional 时，IIS 创建默认的网站和 FTP 站点。本主题描述了如何在这些默认站点上发布信息。

在网站上发布内容
  1. 为网站创建主页。有关创建网站时可使用的工具的详细信息，请参阅选择创作工具。
  2. 将主页文件命名为 Default.htm 或 Default.asp。
  3. 将主页复制到 IIS 的默认 Web 发布目录中。默认 Web 发布目录也称为主目录，安装程序提供的位置是 \Inetpub\Wwwroot。
  4. 如果网络具有名称解析系统（通常为 DNS)，则访问者只要在浏览器地址栏中输入计算机名就可以访问站点。如果网络不具有名称解析系统，访问者必须输入计算机的数字 IP 地址。详细信息，请参阅关于名称解析。
在 FTP 站点上发布内容
  1. 将文件复制或移动到默认的 FTP 发布目录。安装程序提供的默认目录是 \Inetpub\Ftproot。
  2. 如果网络具有名称解析系统（通常为 DNS），那么访问者可以在他们的浏览器地址栏中输入 ftp://，后面跟计算机名，就可以访问该站点。如果网络没有名称解析系统，那么访问者必须输入 ftp:// 和计算机的数字 IP 地址。要自定义 FTP 站点的外观，请参阅设置 FTP 消息和目录输出样式。

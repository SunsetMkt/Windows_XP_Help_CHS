  1. 如何发布文档或网页？

    2. 若要发布文档，请将文件移到 \Inetpub\Wwwroot 目录中。然后在浏览器的地址栏中输入“http://服务器名//文件名”查看发布的文件。有关详细信息，请参阅使用 IIS 快速建立站点。

  2. 如何在网站或 FTP 站点上创建虚拟目录？

    3. 可以使用 Internet 信息服务管理单元在网站上创建虚拟目录。创建虚拟目录中包含有关如何完成此任务的过程信息。

  3. 如何创建网站？

    4. 在 Microsoft® Windows XP 上安装 IIS 时，将为您设置一个默认网站。随后，就可以在默认发布地点发布内容。

  4. 如何创建多个网站？

    5. 安装了 IIS 的 Windows XP Professional 可以在一台计算机上维护一个网站和一个 FTP 站点。如果要在一台计算机上设置多个网站或 FTP 站点，请考虑升级为 Windows Server。

  5. 如何停止和重新启动 Internet 服务？

    6. 多个 Internet 服务运行在一个进程中，并且 Internet 服务通常由其他进程组成。因此，Internet 服务与其他 Windows 服务不同，可以高效地关闭和重新启动。有关使用 IISReset.exe 的详细信息，请参阅重新启动 IIS。

  6. 为什么站点的一些访问者不能获得专门为他们的浏览器设计的网页？

    7. 如果使用浏览器性能组件来确定客户端查看能力，则当新版本的浏览器发布之后，应该定期更新服务器上的文件 Browscap.ini。既可以从 Windows XP CD 盘中获得更新后的 Browscap.ini，也可以从 Microsoft 网站下载。

  7. 在哪里可找到为残疾用户提供的辅助特性信息？

    8. 有关 Windows 辅助功能的常规信息，请参阅 Windows 文档。与 IIS 相关的辅助功能信息，请参阅 IIS 辅助功能。

  8. 如何获得有关 IIS 的技术支持？

    9. 如果已将本软件作为原始设备制造商 (OEM) 软件包的组成部分进行了预安装，或所购买的个人计算机上已包含本软件，则必须与 PC 或服务器制造商联系才能获得技术支持。请查阅 PC 或服务器随带文档获取制造商的电话号码。

    如果以零售形式获得本产品，并且联机文档无法解答遇到的技术问题，请访问网站上的 Microsoft 产品支持服务。

  9. 是否可以更改网站的名称并将对旧站点的请求重新定向到新站点？

    10. 如果网络能够正确地将两种请求路由到您的计算机，则可以正确配置网站并使其对旧名称和新名称均作出相应。这样，访问者使用旧名称也可以访问您的站点，而且还会告之新名称。有关如何分配名称的说明，请参阅命名网站。

  10. 如何才能知道驱动器是否设置为使用 NTFS？

    11. 在“Windows 资源管理器”中，选中驱动器并打开其属性页。在“常规”选项卡上，列出了文件系统类型。

  11. 如何将格式化为文件分配表 (FAT) 文件系统的驱动器或分区转换为 NTFS？

    12. 在命令行下，输入“convert 驱动器号:/FS:NTFS”。NTFS 驱动器不能转换回 FAT 类型。详细信息，请参阅 Windows 文档。

  12. 是否可以设置 FTP 站点使不同用户拥有不同级别的访问权限？

    13. 在 NTFS 文件系统中，可以使用一个虚拟目录并设置 NTFS 权限允许或拒绝不同 Windows 用户帐户进行各种不同的访问。

    如果要根据 IP 地址和域名授权或拒绝访问，应考虑升级到某种 Windows Server 产品。

  13. 如何确认服务器证书是否已被附加到特定网站？

    14. 选择站点并打开其属性页。在“目录安全性”属性页中的“安全通信”下有一个“编辑”按钮。如果此按钮已启用，表明已有证书与网站关联。如果此按钮呈灰色，则表明需要将服务器证书与站点关联才能使用安全套接字层 (SSL) 功能。

  14. 是否可将多个服务器证书附加到一个网站？

    15. 否。每个网站只能与一个服务器证书关联。

  15. 是否可将同一服务器证书附加到多个站点？

    16. 是。一个服务器证书可与任意网站关联。

  16. 是否可将服务器证书附加到一个 FTP 站点？

    17. 否。FTP 站点不支持 SSL 功能。

  17. 是否应该创建服务器证书的副本？

    18. 是。服务器证书非常有用，它是服务器 SSL 功能的核心。若要创建服务器证书的备份，请将整个证书复制到一张软盘上，然后再保存到安全的地方。

  18. 如何知道有多少用户与我的 FTP 服务相连？

    19. 在 Internet 信息服务管理单元中，选择一个 FTP 站点，打开其属性页，然后单击“当前会话”按钮。有关详细信息，请参阅性能调整。

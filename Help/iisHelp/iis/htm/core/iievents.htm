IIS 的 Windows 事件

IIS 在“事件查看器”中生成事件，以便用户验证 IIS 的性能。“事件查看器”跟踪（按重要顺序）错误事件、警告事件和信息事件。“事件查看器”中的日志提供所有 Windows 服务和诸如登录次数和连接次数等进程的审核记录。

本节包含：

  • 关于 Windows 事件：与 Windows 事件相关的主题。
  • Web 服务事件：Web 服务生成的事件列表。
  • FTP 服务事件：FTP 服务生成的事件列表。
  • Active Server Pages 事件：ASP 生成的事件列表。

本节不包含：

  • 客户端浏览器中显示的错误信息；请参阅自定义错误消息。
  • 记录客户端请求；请参阅记录站点活动。

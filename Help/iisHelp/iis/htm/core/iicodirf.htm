为网页添加页脚

可以对 Web 服务器进行配置，自动将 HTML 格式的文件插入到 Web 服务器发送出去的所有网页的底部（不支持对 ASP 网页附加页脚）。例如，可以在文件中包含 HTML 格式的说明，将简单的文字消息和徽标图案添加到网页中。详细信息，请参阅关于网站管理和服务器端包含。

注意   文档页脚可能会降低 Web 服务器的性能，尤其在网页访问频率较高时。

为所有网页添加页脚
  1. 创建一个 HTML 页脚文件，并将其保存在 Web 服务器的硬盘上。

    2. 注意   页脚文件不应是完整的 HTML 文档，应该只包含定义页脚内容外观格式所需的那些 HTML 标记。例如，将公司名称添加到所有网页底部的页脚文件应包含文本和定义文本字体及颜色所必需的 HTML 标记。

  2. 在 Internet 信息服务管理单元中，选择某个网站、目录或文件，并打开其属性页。
  3. 单击“文档”选项卡，然后选中“启用文档页脚”复选框。
  4. 在文本框中，输入页脚文件的完整本地路径，或者使用“浏览”按钮导航到此页脚文件。

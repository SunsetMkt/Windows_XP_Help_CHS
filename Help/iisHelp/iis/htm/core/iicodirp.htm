将请求重定向到程序

作为重定向方法的高级应用，可以将对特定目录中文件的所有请求重定向到某个程序。通常，您还可能希望将原始 URL 中的所有参数也传递到此程序，使用重定向变量可以实现这一目的。

将请求重定向到程序
  1. 在 Internet 信息服务管理单元中，选择某个网站或目录，并打开其属性页。
  2. 单击“主目录”、“虚拟目录”或“目录”选项卡。
  3. 选择“重定向到 URL”。
  4. 在“重定向到”文本框中，输入程序的 URL，包括将参数传递给程序所需的任何重定向变量。

    5. 例如，若要将对脚本目录中脚本的所有请求都重定向到日志记录程序（此程序记录请求的 URL 以及与此 URL 一起传递的任何参数），请输入 /Scripts/Logger.exe?URL=$V+PARAMS=$P。$V 和 $P 是重定向变量。

  5. 选中“上面输入的准确 URL”复选框，防止 Web 服务器将原始文件名追加到目标 URL 中。

有关重定向变量的详细信息，请参阅重定向参考。

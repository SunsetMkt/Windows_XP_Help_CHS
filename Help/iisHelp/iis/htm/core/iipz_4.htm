“目录安全性”属性页

使用此属性页设置 FTP 站点的安全功能。

匿名访问和身份验证控制

如果要配置 FTP 站点的身份验证和匿名访问功能，请单击“编辑”。使用这些功能将 FTP 站点配置为在授权访问受限制的内容之前确认用户标识。

不过，在 FTP 站点可以验证用户身份之前，必须首先创建有效的 Windows 用户帐户，然后为这些帐户配置 Windows 文件系统 (NTFS) 目录和文件权限。

IP 地址和域名限制

如果要允许或防止特定的用户、计算机、计算机组或域访问此 FTP 站点或目录，请单击“编辑”。

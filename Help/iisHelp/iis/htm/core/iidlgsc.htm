设置默认登录域

通过“基本身份验证”进行验证的用户必须使用有效的 Windows 用户名和密码登录。用户名通常包括：

  • Windows 域名。
  • 帐户用户名。

上述信息用来识别被 Web 服务器作为单一实体管理的计算机或计算机组。例如，使用 PhilSpencer 帐户登录 Sales1 域的用户将被登录为 Sales1\PhilSpencer。然而，如果用户不提供域名，您可以对 Web 服务器进行配置，为“基本身份验证”假定一个默认登录域。

设置默认登录域
  1. 在 Internet 信息服务管理单元中，选择某个网站、目录或文件，并打开其属性页。
  2. 选择相应的“目录安全性”或“文件安全性”属性页。在“匿名访问和身份验证控制”下，单击“编辑”。
  3. 在“验证方法”对话框中，选中“基本身份验证”复选框。

    4. 注意   由于基本身份验证以未加密形式在网络上发送密码，因此将出现一个对话框，询问是否希望继续。单击“是”继续。

  4. 单击“编辑”。
  5. 在“基本身份验证”对话框中，输入或者浏览到新的默认登录域。单击“使用默认值”使用 Web 服务器的默认域名。

自定义错误消息

可以配置 IIS 发送自定义错误消息，而不是发送默认的 HTTP 1.1 错误消息。可以将这些自定义错误消息映射为文件名或 URL。

本节包含：

  • 关于自定义错误消息：有关默认和自定义 HTTP 1.1 错误的概述信息。
  • 添加自定义错误消息：有关如何向网站添加自定义错误消息的过程信息。
  • 启用详细自定义错误消息：有关如何使用 IIS 附带的详细（或友好） HTTP 1.1 错误信息的过程信息。
  • 启用 ASP 错误处理：有关如何使用 500-100.asp 自定义错误文件处理 ASP 错误的概述信息。

本节不包含：

  • 网站内容；请参阅网站管理。
  • Windows 事件；请参阅 IIS 的 Windows 事件。

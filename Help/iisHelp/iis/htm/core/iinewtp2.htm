管理

本节说明如何使用 Internet 信息服务管理单元来管理 IIS。Internet 信息服务管理单元驻留在 Microsoft Management Console (MMC) 中，是一个强大的站点管理工具，可提供对所有服务器设置的访问。使用 Internet 信息服务管理单元可管理企业 Intranet 中的复杂站点，或在将站点内容上载到 Internet 信息服务器之前管理模拟调试的站点。

本节包含：

  • 网站管理：创建站点内容并创建必要的发布目录结构。
  • 服务器管理：有关调整默认设置以提高性能或改进安全性的高级主题。
  • 管理员参考：有关注册表和配置数据库设置的详细信息。

注意   Internet 信息服务管理单元是用于 IIS 的管理工具，它已经和 Windows XP 的其他管理功能集成。在以前的版本中，此工具称为 Internet 服务管理器。

本节不包含：

  • 有关建立网站或 FTP 站点的信息；请参阅使用 IIS 快速建立站点。
  • 有关安装或安装程序的信息；请参阅安装 IIS。

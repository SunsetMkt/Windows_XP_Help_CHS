服务器端包含

“服务器端包含 (SSI)”指令指示 Web 服务器将另一文件的内容插入网页。使用包含文件，可以将多个网页所使用的信息存储到一个文件中。例如，可以使用 SSI 命令将版权声明插入网站中的所有文档中。

通过 Internet Explorer 5.0 发布

在 Windows XP、Windows 2000、Windows NT 4.0、Windows 98 或 Windows 95 操作系统上，可以通过 Internet Explorer 5 与 WebDAV 目录进行连接。建立连接之后，就可以处理文件并将文件发布到此目录，与通过 Windows 2000 建立连接的效果一样。

通过 Internet Explorer 连接 WebDAV 目录
  1. 从 Internet Explorer 5.0 的“文件”菜单，单击“打开”。
  2. 在“打开”框中输入希望与之建立连接的 WebDAV 目录的 URL。
  3. 选中“以 Web 文件夹方式打开”框，再单击“确定”。将显示此目录下的文件。

Internet 信息服务管理单元中超时设置的位置

此页是对超时表主题的补充。

更改 AspScriptTimeout 值
  1. 在 Internet 信息服务管理单元中，选择网站或应用程序的开始位置目录。
  2. 打开目录的属性页，然后单击“主目录”、“虚拟目录”或“目录”选项卡。
  3. 单击“配置”按钮，然后单击“应用程序选项”选项卡。
  4. 在“ASP 脚本超时”中，输入期望值。

--或者--
Internet 信息服务管理单元|属性|配置按钮|应用程序选项|ASP 脚本超时
更改 AspSessionTimeout 值
  1. 在 Internet 信息服务管理单元中，选择网站或应用程序的开始位置目录。
  2. 打开目录的属性页，然后单击“主目录”、“虚拟目录”或“目录”选项卡。
  3. 单击“配置”按钮，然后单击“应用程序选项”选项卡。
  4. 在“会话超时”文本框中，输入期望值。
更改 ConnectionTimeout 值
  1. 在 Internet 信息服务管理单元中，选择网站或应用程序的开始位置目录。
  2. 打开目录的属性页，然后单击“网站”选项卡。
  3. 在“连接超时”文本框中，输入期望值。
更改 CGITimeout 值
  1. 在 Internet 信息服务管理单元中，选择网站或应用程序的开始位置目录。
  2. 打开目录的属性页，然后单击“主目录”、“虚拟目录”或“目录”选项卡。
  3. 单击“配置”按钮，然后单击“进程选项”选项卡。
  4. 在“CGI 脚本超时”文本框中，输入期望值。
记录网站的处理器使用情况
  1. 在 Internet 信息服务管理单元中，选择要记录其处理器使用情况的网站。
  2. 打开网站的属性页，然后单击“性能”选项卡。
  3. 选择“启用进程限制”并输入限制此站点占用 CPU 时间的百分比。
  4. 如果希望启用 CpuLimitPriority 和 CpuLimitProcStop，请选择“强制性限制”。

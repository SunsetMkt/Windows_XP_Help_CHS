证书

证书是服务器或从服务器请求信息的客户端的一种数字标识，其功能与护照或其他标识持有者身份的官方证件相似。证书是 IIS 安全套接字层 (SSL) 功能的一部分，通过安全套接字层，可以建立安全连接以发送敏感信息。

本节包含：

  • 关于证书：证书的概念性概述。
  • 在服务器上设置 SSL：概述如何在服务器上设置和管理安全套接字层 (SSL)。
  • 使用新的安全任务向导：证书、CTL 和权限向导中可用功能的概念性概述。
  • 获取服务器证书：了解如何获取使用 SLL 所需的服务器证书。
  • 使用证书信任列表：概述如何将新的证书颁发机构添加到服务器的可信任颁发机构列表中。
  • 获取客户证书：概述如何获取称为客户证书的用户数字标识。
  • 启用客户证书：概述启用客户证书 SLL 身份验证的过程。
  • 将客户证书映射到用户帐户：概述如何使用客户证书将用户直接连接到 Windows 用户帐户。

本节不包含：

  • 控制用户访问 Web 服务器文件及目录的方法；请参阅访问控制。
  • 有关设置或更改加密方法的信息；请参阅加密。

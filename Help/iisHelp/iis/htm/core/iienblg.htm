启用日志记录

可以针对各个网站和 FTP 站点启用日志记录并选择日志格式。启用日志记录时，会对所有网站的目录启用日志记录，但是可以对特定目录禁用日志记录。

启用网站或 FTP 站点上的日志记录。
  1. 选中某个网站或 FTP 站点，并打开其属性页。
  2. 在“网站”或“FTP 站点”属性页中，选中“启用日志记录”复选框。
  3. 在“活动日志格式”列表中，选择一种格式。默认情况下，“启用日志记录”复选框被选中，其格式为“W3C 扩展日志文件格式”，并已启用了下列字段：“时间”、“客户端 IP 地址”、“方法”、“URI 源”以及“HTTP 状态”。
  4. 单击“应用”。
  5. 单击“确定”。
禁用或启用网站上特定目录的日志记录
  1. 选择目录并打开其属性页。
  2. 在“主目录”或“目录”属性页中，查看“记录访问”复选框。默认情况下，此复选框被选中。
  3. 要禁用目录的日志记录，请清除此复选框；要启用日志记录，请选中此复选框。这些设置不会影响进程计帐。

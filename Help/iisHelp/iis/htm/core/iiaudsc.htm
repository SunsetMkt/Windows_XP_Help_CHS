审核

可以使用安全审核技术跟踪用户活动，并侦测未经授权用户对 NTFS 目录和文件的访问尝试。可以审核的活动包括：

  • 所有登录尝试（成功和不成功的）。
  • 访问限制帐户的用户尝试。
  • 执行限制命令的用户尝试。

本节包含：

  • 关于审核：了解如何监视和评估 Web 服务器的安全性。
  • 配置和监视审核：创建安全日志。
  • 侦测未授权的访问：了解如何解释安全日志以识别未经授权的访问。

本节不包含：

  • 如何记录用户网站和 FTP 站点的活动；请参阅记录站点活动。
  • 如何控制对网站和 FTP 站点的访问；请参阅访问控制。

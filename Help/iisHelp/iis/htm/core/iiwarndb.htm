配置应用程序

本节说明如何使用 Internet 信息服务管理单元配置基于网站的应用程序。

本节包含：

  • 关于应用程序：定义基于网站的应用程序，并解释站点目录结构如何与应用程序的定义相关。
  • 创建应用程序：将目录设置为应用程序的开始位置或根。
  • 设置应用程序映射：使文件扩展名与处理这些文件的程序相关联。
  • 独立应用程序：在 Web 服务器内存空间之外的独立内存空间中运行应用程序。
  • 缓存应用程序：设置 CGI 脚本超时，缓存 ISAPI DLL 以及更改 ASP 网页的缓存。
  • 终止独立的应用程序：终止独立的应用程序。
  • 进程回收：对回收应用程序的方法、可配置事件、编程和设置注意事项进行了说明。
  • 配置 ASP 应用程序：为 ASP 应用程序设置属性。
  • 启用 ASP 调试：在 Web 服务器上启用调试，以便可以检查和改正 ASP 网页。
  • 配置 CGI 应用程序：在 Web 服务器上安装和配置 CGI 应用程序。
  • 安装 ISAPI 筛选器：解释什么是 ISAPI 筛选器以及如何使其与 Web 服务器事件相关联。

本节不包含：

  • 编写 ASP 应用程序；请参阅 Active Server Pages。
  • 使用证书；请参阅证书。

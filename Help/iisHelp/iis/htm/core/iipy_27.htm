多网站高级配置

使用此对话框配置网站的附加标识。

单击“添加”或“删除”按钮可以添加或删除列出的标识。要编辑现有的标识，请选择 IP 地址并编辑相应的字段“IP 地址”、“TCP 端口”、“SSL 端口”或“主机头名”。

每个网站必须具有唯一的标识特性组合。因此，当多个网站的三个标识特性（域或主机头名、IP 地址和端口）中的两个相同时，另一个特性必须不同。而且，由于 SSL 证书包含域名，所以使用证书的网站不能与其他网站共享 IP 地址。

SSL 端口

确定用于建立安全套接字层 (SSL) 连接的端口。

对于 Web 服务器的默认网站，此端口设置为 443。如果创建新网站，必须明确设置此端口。可以将 SSL 端口更改为任意唯一的 TCP 端口号；不过，用户必须事先知道应请求此端口号。否则，他们的请求将不能连接到服务器。只有安装了服务器证书之后，此选项才可用。

访问控制

可以控制允许哪些用户和哪些计算机访问您的 Web 服务器及其资源。既可以使用 NTFS 安全功能，也可以使用 Web 服务器安全功能为用户分配对某些目录和文件的特定访问权限。

本节包含：

  • 关于访问控制：有关如何限制访问某些文件和目录的概述。
  • 使用 NTFS 保护文件安全：有关如何配置服务器使用 NTFS 安全功能的过程信息。
  • 设置目录或文件的 NTFS 权限：有关如何配置 NTFS 权限的过程信息。在此过程中，将定义为 Windows 用户帐户分配的文件或目录访问级别。
  • 设置 Web 和 FTP 权限：设置将应用于所有用户的 Web 和 FTP 内容限制。
  • 控制数据库访问：有关如何防止未经授权用户访问数据库文件的指南。

本节不包含：

  • 配置内容分级特性，在用户试图访问无权访问的 Web 内容时发出警告；详细信息，请参阅使用内容分级。

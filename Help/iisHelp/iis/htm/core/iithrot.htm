限制带宽

如果其他服务（如电子邮件）也在使用 Web 服务器所使用的网络连接，可能需要对 Web 服务器所使用的带宽进行限制，以便使其他服务也可以使用带宽。可以只限制静态 HTML 文件使用的带宽。

确定服务器使用的带宽

  1. 使用系统监视器检查 Network Interface 对象中的 Bytes Total/sec 或 Current Bandwidth 计数器。（如果希望比较传入和传出通信量，可以检查 Bytes Sent/sec 和 Bytes Received/sec。）
  2. 将这些值与网络连接的总带宽进行比较。在负载正常的情况下，服务器使用的带宽应该不超过全部可用带宽的 50%。如果服务器经常出现负载高峰，甚至尖峰，则最好将正常负载保持在总量的 50% 以下，将剩余的“净空”带宽留在高峰期间使用。
限制 IIS 使用的带宽
  1. 在 Internet 信息服务管理单元中，选择运行 IIS 的计算机。
  2. 在“Internet 信息服务”属性页中，选中“启用带宽限制”复选框。
  3. 在“最大网络使用”框中，以 kb/s 为单位输入希望 IIS 使用的最大带宽。

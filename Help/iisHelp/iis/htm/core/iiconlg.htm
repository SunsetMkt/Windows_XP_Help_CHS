自定义 W3C 扩展日志记录

如果在网站或 FTP 站点上使用 W3C 扩展日志文件格式，可以通过选择要记录的字段（项）进行自定义。可以使用那些对您很重要的字段收集详细信息，而略去不需要的字段以限制日志文件的大小。

自定义 W3C 扩展日志记录
  1. 选择某个网站或 FTP 站点，打开其属性页。
  2. 如果未启用日志记录，请启用，并选择 W3C 扩展日志文件格式。
  3. 单击“属性”。
  4. 在“扩展属性”属性页上，选择要记录的字段。
  5. 单击“应用”。

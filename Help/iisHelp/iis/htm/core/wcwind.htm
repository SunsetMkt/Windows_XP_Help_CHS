通过 Windows XP 发布

通过 Windows XP Professional 可以与另一台服务器上的 WebDAV 发布目录连接。默认情况下，Windows XP Professional 上不会安装 IIS 5.1。因此，如果没有在安装选项中选中 IIS 5.1，则必须进行手动安装。要安装 IIS，请参阅安装 IIS。

通过 Windows XP 与 WebDAV 目录连接
  1. 单击“开始”，单击“我的电脑”，再单击“网上邻居”。
  2. 双击“添加网上邻居”。
  3. 在“添加网上邻居向导”中，输入要连接 WebDAV 目录的 URL。
  4. 单击“下一步”。
  5. 输入要连接目录的友好名称。
  6. 单击“完成”。
  7. 双击“网上邻居”，会出现 WebDAV 站点的友好“Web 文件夹”名称。例如，如果输入 My Publishing Folder 作为友好名称，将会出现 My Publishing Folder 的快捷方式。
  8. 双击 Web 文件夹名称。
  9. 双击 My Publishing Folder 的快捷方式，将显示此目录中的文件。

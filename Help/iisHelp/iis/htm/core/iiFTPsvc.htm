FTP 服务

安装 IIS 5.1 时将默认安装 FTP 服务。安装 FTP 服务时，将在 x:\Inetpub\Ftproot 中创建一个默认 FTP 站点，其中 x: 表示安装 IIS 的驱动器号。如果安装了 IIS 但卸载了 FTP 服务，可以通过下列步骤重新安装 FTP 服务。

安装 FTP 服务
  1. 在“控制面板”中，单击“添加或删除程序”。
  2. 单击“添加/删除 Windows 组件”。
  3. 从列表中选择“Internet 信息服务 (IIS)”，然后单击“详细信息”。
  4. 选中“文件传输协议 (FTP) 服务”复选框。
  5. 单击“确定”（可能会提示您输入 Windows XP CD 或网络安装路径名）。
  6. 单击“下一步”。
  7. 单击“完成”。

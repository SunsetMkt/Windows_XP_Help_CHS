获取客户证书

可以从值得信任的、称为证书颁发机构的第三方机构获取客户证书。颁发证书之前，颁发机构可能需要验证您的身份。根据您希望获得的证书类型的不同，颁发机构要求的标识信息也有所不同。如何获取和实施这些证书，取决于所使用的浏览器。此主题中的信息仅适用于 Internet Explorer 3.0 和更高版本。有关如何获取和实施其他浏览器的客户证书，请参阅浏览器文档。

获取客户证书
  1. 选择证书颁发机构。选择证书颁发机构时，请考虑下列因素：
    • 此证书颁发机构是否知名并且可信？
    • 为了验证您的身份，颁发结构需要您提供哪些信息？大多数证书颁发机构需要您提供详细的标识信息，例如姓名、地址、单位名称和电子邮件地址等。
    • 证书颁发机构是否具有网站可供用户申请客户证书？
    • 颁发的证书与所使用的浏览器和服务器是否兼容？

    有关为 Internet 信息服务提供证书颁发服务的证书颁发机构的列表，请参阅获取服务器证书。

  2. 参阅证书颁发机构提供的特定订购说明。
  3. 订购证书。某些客户证书只对生成证书请求的计算机和 Web 浏览器有效，请确保从将要安装证书的计算机上联机发出申请。
  4. 收到客户证书后，使用 Web 浏览器的安全功能安装此证书。（有些 Web 浏览器可能会将客户证书视为“浏览器”或“个人”证书。）

“文档”属性页

使用此属性页定义站点的默认网页并将页脚附加到站点的文档中。

启用默认文档

如果选中此复选框，当浏览器请求未指定文档名称时，将提供此默认文档。默认文档可以是目录主页或包含站点文档目录列表的索引页。

可以在文本框中输入文档名称，添加新的默认文档。可以指定多个默认文档。按照在列表中显示的名称顺序依次提供默认文档。服务器返回找到的第一个文档。如果指定多个默认文档，文档名称之间要用逗号分隔。

启用文档页脚

可以使用此选项在 Web 服务器发送的每个文档中自动附加 HTML 格式的页脚。

页脚文件不应该是完整的 HTML 文档。它只应包含格式化页脚内容的外观和功能所必需的 HTML 标记。

要指定页脚文件的完整路径和文件名，可单击“浏览”。

“HTTP 头”属性页

使用此属性页设置返回到浏览器的 HTTP 网页头中的值。

启动内容过期

选中此复选框以包含过期日期信息。应在时间敏感的内容（例如特别优惠或事件公告）中包含日期。浏览器将当前日期与过期日期相比较，确定是显示缓存中的网页还是从服务器请求更新的网页。

自定义 HTTP 头

可以使用此属性将自定义 HTTP 头从 Web 服务器发送到客户端浏览器。例如，可以使用自定义 HTTP 头允许客户端浏览器将页面放入缓存而防止代理服务器缓存页面。自定义 HTTP 头在配置数据库中描述。

要使 Web 服务器发送 HTTP 头，请单击“添加”，然后在文本框输入 HTTP 头的名称和值。

要停止发送 HTTP 头，请选中它并单击“删除”。

内容分级

使用内容分级可以在网页的 HTTP 头中嵌入描述性的标注。浏览器（例如 Microsoft Internet Explorer 3.0 或更高版本）可以检测内容分级，并帮助用户识别潜在的令人反感的 Web 内容。

要为网站、目录或文件设置内容分级，请单击“编辑”。

MIME 映射

可以单击“编辑”按钮配置多用途网际邮件扩充协议 (MIME) 映射。这些映射设置 Web 服务返回给浏览器的不同文件类型。

如果在主属性页中为计算机设置 MIME 映射，那么计算机上的网站和目录将使用同一映射。可以修改一个网站或目录的 MIME 映射。但是，如果重新应用主属性，则已修改的网站或目录的属性将被主属性完全替换。也就是说，属性并不会合并。

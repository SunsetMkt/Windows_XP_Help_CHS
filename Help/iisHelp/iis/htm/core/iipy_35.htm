“进程选项”属性页

使用此属性页更改在单个进程中运行的应用程序的设置。可以使用此属性页更改两种类型进程的设置：Web 服务器进程和单独的应用程序进程。网站选项设置影响在 Web 服务器进程中运行的所有应用程序。单独应用程序的选项设置只影响此应用程序本身。

将不成功的客户端请求写入事件日志

选中此复选框可以使 Web 服务器将不成功的客户端请求写入 Windows 事件日志。

启用调试异常捕获

如果选中此复选框，当 ASP 无法创建 ActiveX 组件（对 Server.CreateObject 的调用）提供的对象例程时，ASP 将发送一般错误消息。未选中此复选框时，ASP 将传送来自于组件的特定错误消息。调试组件时撤消此选项。完成调试后，重新选中此选项。

缓存的脚本引擎数

指定 Active Server Pages 在缓存中保留的 ActiveX 脚本引擎的最大数目。

脚本文件缓存选项

使用此选项指定缓存在内存中的预处理的 .asp 文件数。增加缓存的文件数可以改善 Active Server Pages 的性能。要缓存所有 .asp 文件，可选中“缓存所有请求的 ASP 文件”。要指定缓存脚本文件的最大数目，可选中“在内存中缓存的 ASP 文件数”，并在文本框中输入值。

CGI 脚本超时

可以设置或更改超时值，它是 CGI 脚本用来在操作停止前执行并返回值的时间。

注意   可以在网站、虚拟目录和目录级别上设置应用程序配置属性。

加密

可以使用 Web 服务器的 SSL 加密功能来编码 Web 服务器传输和接收的信息。当启用 SSL 加密时，可以防止未授权个人解码传输的原始内容。

本节包含：

  • 关于加密：SSL 安全通信概述。了解加密如何保护通过不安全网络传输的信息。
  • 启用加密：启用 Web 服务器 SSL 加密功能的步骤。
  • 设置加密强度：了解如何要求用户与 Web 服务器之间建立 128 位会话密钥的安全通信。

本节不包含：

  • 创建用于 SSL 身份验证的 Web 服务器密钥对文件；请参阅使用新的安全任务向导。
  • 在使用 SSL 安全功能之前，首先应获得 Web 服务器证书；请参阅获取服务器证书。

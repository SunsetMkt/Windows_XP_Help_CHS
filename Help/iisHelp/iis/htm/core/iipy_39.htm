分级属性页

使用此属性页对特定网站、目录或文件配置 Web 服务器的内容分级功能。

Web 服务器的“Internet 内容选择平台 (PICS)”系统使用 Recreational Software Advisory Council (RSAC) 开发的分级系统，根据暴力、裸露程度、色情程度以及攻击性语言的级别对内容分级。

要对 Web 内容应用内容分级，可选中“对资源启用分级”复选框。

要设置内容分级，可在“类别”框中单击分级类别（语言、裸露、性或暴力）。然后使用分级标尺，设置要禁止何种级别的内容。每项设置都显示有对分类级别的说明。

要标识对内容进行分级的人，可在“内容分级人员的电子邮件”文本框中输入其电子邮件地址。

要设置分级内容的过期日期，可单击“过期日期”框右边的按钮。将出现日历控件，可用来指定过期日期。

发布和管理文件

本节讲述如何与 WebDAV 发布目录连接。如何将文档从所在的计算机拖放到发布目录中进行发布以及如何处理发布目录中的文件。

本节包含：

  • 通过 Windows XP 发布：介绍如何使用“网上邻居”中的“Web 文件夹”功能与 WebDAV 服务器连接。
  • 通过 Internet Explorer 5.0 发布：介绍如何在 Internet Explorer 5.0 中连接到 WebDAV 服务器并复制文件。
  • 通过 Office 2000 发布：介绍如何在 WebDAV 虚拟目录中直接管理和编辑文件。
    •

注意   即使用户从防火墙后面连接，在具有适当权限并且防火墙配置为允许发布的情况下，用户仍可以在 WebDAV 目录中发布文档。

命名网站

每个网站（虚拟服务器）都具有一个描述性名称，并支持一个或多个主机头名。主机头名使在一台计算机上维护多个域名成为可能。并不是所有浏览器都支持主机头名。Microsoft Internet Explorer 4.0 和 Netscape Navigator 4.0 浏览器及其最新版本均支持使用主机头名，但这两种浏览器的早期版本不支持。

如果访问者试图使用不支持主机头的旧版本浏览器连接您的站点，则访问者将被定向到此 IP 地址对应的默认网站（如启用了默认站点），此站点不必是被请求的站点。另外，如果浏览器请求的站点目前停止使用，则访问者将接收到默认网页。因此，必须仔细斟酌默认网站显示的内容。通常，ISP 将其主页作为默认显示网页，而不是某个客户的网站。这可防止将对已停止站点的请求转到错误的站点。此外，默认站点可以包含一个支持旧版本浏览器使用主机头名的脚本。

命名网站
  1. 在 Internet 信息服务管理单元中，选择网站并打开其属性页。
  2. 在“网站”属性页的“描述”文本框中，输入此站点的描述性名称。
分配主机头名
  1. 询问系统管理员，将映射到静态 IP 地址的新站点名称添加到名称解析系统（通常是 DNS）。
  2. 在“网站”属性页中，单击“高级”按钮。
  3. 在“多网站高级配置”属性页中，单击“添加”按钮分配网站的主机头文件、IP 地址以及端口。

    4. 注意   使用安全套接字层 (SSL) 时主机头不起作用，因为主机头包含在加密的客户端请求中。但是，每台服务器可以有多个服务器证书、多个 IP 地址及多个 SSL 端口。

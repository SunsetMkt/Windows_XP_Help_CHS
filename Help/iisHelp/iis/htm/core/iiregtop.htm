注册表

配置注册表中所存储的值定义了 Windows 操作系统以及在运行 Windows 的计算机上安装的所有服务的工作环境。可以使用图形工具，例如“控制面板”、“Windows 安装程序”或 Internet 信息服务管理单元来更改这些值，也可以使用 Windows 中所包含的注册表编辑器 (Regedt32.exe) 工具直接修改配置注册表。

本节包含：

  • 关于注册表：与注册表相关的概念。
  • 全局注册表项：用于 Internet 服务全局控制的参数。
  • 服务特定的注册表项：通过 Internet 服务保存在注册表中的参数，用于服务特定的行为。
  • WWW 服务注册表项：仅用于 WWW 服务的参数。
  • FTP 服务注册表项：仅用于 FTP 服务的参数。

本节不包含：

  • 配置数据库；请参阅配置数据库。
  • 例行服务器管理；请参阅服务器管理。

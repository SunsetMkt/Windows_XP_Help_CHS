Internet 服务管理器 (HTML)

Internet 服务管理器的 HTML 版本在浏览器中具有拆分式屏幕布局，它包括浏览器导航窗格中的“菜单视图”和浏览器内容窗格中的“树视图”。

“树视图”显示可管理的所有计算机、网站、FTP 站点、虚拟目录和目录。选择树视图中的任何项目后，它将在浏览器中突出显示。当项目突出显示时，可以从“菜单视图”选择命令，此命令将影响突出显示的树视图项目。

“菜单视图”列出如下命令：

新建
新建网站或 FTP 站点。

删除
删除网站、FTP 站点、虚拟目录或目录。

重命名
重命名网站、FTP 站点或虚拟目录。

属性
显示所选树视图项目的属性。

启动
启动选定的网站或 FTP 站点。

停止
停止选定的网站或 FTP 站点。

暂停
暂停选定的网站或 FTP 站点。

恢复
恢复选定的网站或 FTP 站点。

权限向导
设置 Web 和 FTP 权限、访问权限和身份验证方案。

主属性
设置计算机上所有当前（或新的）网站和 FTP 站点使用的默认值。

备份配置
保存所有在 Internet 服务管理器 (HTML) 会话中配置的设置。

警告   当使用不安全的连接时，使用 Internet 服务管理器 (HTML) 的唯一安全方法是通过安全套接字层 (SSL)。

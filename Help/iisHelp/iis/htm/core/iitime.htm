超时参考

Internet 信息服务具有几个超时值，在安装 Windows XP 时已对它们进行了默认设置。使用超时值，服务器可以指定将服务器资源分配给特定任务或客户端使用的时间。下表列出了 IIS 中的不同超时值及其设置位置。

属性名称 配置数据库标识符 在 Internet 信息服务管理单元中是否可用
AspScriptTimeout MD_ASP_SCRIPTTIMEOUT 是
AspSessionTimeout MD_ASP_SESSIONTIMEOUT 是
ConnectionTimeout MD_CONNECTION_TIMEOUT 是
CGITimeout MD_SCRIPT_TIMEOUT 是
ServerListenTimeout MD_SERVER_LISTEN_TIMEOUT 否
PoolIdcTimeout MD_POOL_IDC_TIMEOUT 否
AspQueueTimeout MD_ASP_QUEUETIMEOUT 否
CpuLimitLogEvent MD_CPU_LIMIT_LOGEVENT 是
CpuLimitPriority MD_CPU_LIMIT_PRIORITY 是
CpuLimitProcStop MD_CPU_LIMIT_PROCSTOP 是
CpuLimitPause MD_CPU_LIMIT_PAUSE 否

IIS 重新启动超时 默认值（秒）
停止 60
重新启动 20
重新引导 0

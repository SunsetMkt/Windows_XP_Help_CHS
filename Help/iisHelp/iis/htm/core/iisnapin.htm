打开 Internet 信息服务管理单元

Internet 信息服务管理单元由 Microsoft Management Console (MMC) 维护，此管理单元提供的服务器管理选项可以控制网站或 FTP 站点的内容及对这些站点的访问。在以前的版本中，此工具称为 Internet 服务管理器。

打开 Internet 信息服务管理单元


  1. 在“开始”菜单上，单击“运行”，出现“运行”对话框。
  2. 在“打开”文本框中，输入“inetmgr”并按 Enter 键，出现“Internet 信息服务”管理单元。
还可以通过“计算机管理”对话框访问 IIS。以这种方式访问 IIS 时，无法使用 Internet 信息服务管理单元提供的全部管理选项，但可以快速访问网站并使用有限的管理选项。
  1. 用鼠标右键单击“我的电脑”，单击“管理”，出现“计算机管理”对话框。
  2. 在左窗格中，展开“服务和应用程序”目录树。
  3. 单击“Internet 信息服务”。在右窗格中，将显示网站的名称和状态。

    4. 注意   展开左窗格中的“Internet 信息服务”目录树以及出现的任意网站目录树，可以查看此网页的目录和虚拟目录列表。

有关如何使用 Internet 信息服务管理单元以及管理 IIS 的详细信息，请参阅管理。

IIS 辅助功能

下列快捷键方法（或快捷键命令）是 IIS 特有的。要查看 Windows 所有快捷键的完整列表，请在 Windows 帮助的索引中查找“‘开始’菜单：键盘命令”。要了解有关在 Microsoft Management Console (MMC) 中导航的特定信息，请在 Windows 帮助索引中查找“MMC：辅助功能”。

访问 Internet 信息服务管理单元
  1. 单击“开始”，再单击“运行”，出现“运行”对话框。
  2. 输入 mmc %systemroot%\system32\inetsrv\iis.msc。
  3. 按 Enter 键，出现“Internet 信息服务”管理单元。

注意   如果没有出现 Internet 信息服务管理单元，请检查输入的命令是否正确。mmc 和百分号之间应有一个空格。

还可以从“计算机管理”窗口访问 IIS。以这种方式访问 IIS 时，无法选择 Internet 信息服务管理单元提供的管理选项，但可以快速访问网站并执行受到一定限制的管理。

  1. 单击“开始”，用鼠标右键单击“我的电脑”，从下拉列表中单击“管理”，出现“计算机管理”窗口。
  2. 在左窗格中，展开“服务和应用程序”目录树。
  3. 单击“Internet 信息服务”。在右窗格中，将显示网站的名称和状态。

注意   展开左窗格中的“Internet 信息服务”目录树以及出现的任意网站目录树，可以查看此网页的目录和虚拟目录列表。

访问特定的属性页
  1. 在 Internet 信息服务管理单元中，可以使用方向键选择要管理的站点、虚拟目录或文件。要访问文件，请转到详细信息窗格。在 Internet 信息服务管理单元中，可以使用快捷键 ALT 在控制台树和详细信息窗格两者之间切换。
  2. 按快捷键 Shift+F10，打开“操作”菜单选择项目。

    3. 注意   可以使用快捷键 Ctrl+V 打开“查看”菜单。

  3. 使用向上键或向下键选择“属性”，再按 Enter 键。

    4. 注意   输入 P 将暂停站点。

  4. 按快捷键 Tab，直到选定了属性页标题。
  5. 使用向左键或向右键在同一行中选择相应的属性页。
  6. 要切换到其他属性页行，请使用向上键或向下键。
  7. 要使用某一属性页，请按 Tab 键。可以使用快捷键 Tab 选择项目，或使用快捷键 Alt+<带下划线的字母>。

    8. 注意   要选中或清除某一复选框，请选中它并按空格键。

导航 IIS 联机文档
  1. 如果“目录”选项卡没有显示在最前面，请按快捷键 Tab 或 Shift+Tab，直到突出显示“目录”选项卡为止。
  2. 按 Enter 键，将某一选项卡置于最前面。

    3. 注意   通过这种方法，可以在“目录”、“搜索”和“索引”选项卡之间切换。

  3. 按 Tab 快捷键，直到突出显示了目录中的某个项目。
  4. 使用向上键或向下键选择项目。
  5. 要打开一本书，请使用向右键。
  6. 要关闭一本书，请使用向左键。
  7. 要打开一个文档，请选中它并按 Enter 键。

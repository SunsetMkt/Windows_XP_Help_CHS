服务器管理

欢迎访问有关服务器管理的主题。本节描述了如何更改默认设置，以及如何记录站点通信和监视服务器性能。

可以在 IIS 中使用默认设置，也可以自定义这些设置以适应自己的 Web 发布需要。通过调整默认设置，可以提供附加值、更好的性能以及更高的安全性。

本节包含：

  • 网站和 FTP 站点：有关网站和 FTP 站点的基本信息，包括操作指导、远程管理和命名站点。
  • WebDAV 发布：用 WebDAV 向网站发布信息，是对 HTTP 1.1 提议草案的实现。
  • 名称解析：设置 WINS 或 DNS 名称解析系统，以便将“友好”名称正确映射到 IP 地址。
  • 配置应用程序：控制应用程序作用域以获得最佳的可靠性和性能。
  • 安全性：保护服务器、内容和应用程序不受非法入侵。
  • 自定义错误消息：为用户提供对用户友好的自定义 HTTP 1.1 错误消息。
  • 记录站点活动：跟踪站点通信并解释日志文件。
  • 性能调整：监视并提高服务器性能。
  • 管理脚本：使用命令行脚本管理服务器。

本节不包含：

  • 虚拟目录；请参阅网站管理。
  • 脚本；请参阅 Active Server Pages。

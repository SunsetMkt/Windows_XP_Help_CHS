管理员参考

欢迎浏览“管理员参考”。本主题提供有关注册表配置属性、配置数据库、IIS 的 Windows 事件、计数器、超时和日志记录属性的详细信息，旨在为管理任务提供参考。

本节包含：

  • 注册表：IIS 注册表项的配置属性。
  • 配置数据库：配置数据库的配置属性。
  • IIS 的 Windows 事件：IIS 记录到 Windows 事件日志中的事件列表。
  • 其他参考信息：有关计数器信息、超时和日志记录属性默认设置的参考。

本节不包含：

  • 例程服务器管理；请参阅服务器管理。
  • IIS Admin 对象；请参阅使用 IIS Admin 对象。
  • 管理脚本；请参阅管理脚本。

“操作员”属性页

使用此属性页定义哪些 Windows 用户帐户对此站点拥有操作员权限。

网站操作员

要添加用户帐户到操作员列表，单击“添加”。

要删除当前选定的用户帐户，单击“删除”。

操作员可以执行有限的网站管理功能，包括：

  • 设置 Web 服务器访问权限和日志。
  • 更改默认 Web 文档和页脚。
  • 启用 Web 内容过期、HTTP 头和内容分级功能。

具有操作员权限的用户不能执行下列管理功能，如：

  • 更改网站标识。
  • 重新配置匿名用户名和密码。
  • 限制带宽。
  • 创建虚拟目录或更改虚拟目录路径。
  • 选择或取消选择应用程序隔离。

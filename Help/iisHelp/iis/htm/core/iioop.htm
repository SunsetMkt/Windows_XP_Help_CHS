进程回收

本节讲解的内容包括进程回收、它的可配置事件、为什么要按不同的配置数据库级别配置设置，以及如何继承配置设置。本节的末尾是有关设置注意事项的讨论。

本节包含：

  • 关于进程回收：进程回收的概述、它能解决的问题、回收应用程序的方法以及对可配置事件的介绍。
  • 级别与继承：描述可供配置进程回收的配置数据库的级别，以及配置设置是如何被继承的。
  • 配置回收事件：描述如何配置进程回收事件。
  • 设置注意事项：描述全新安装 Windows 2000 与升级到 Windows 2000 两种情况之间，默认配置数据库关键字的区别。

本节不包含：

  • 配置应用程序；请参阅配置应用程序。
  • 配置数据库属性列表；请参阅字母顺序属性列表。

恢复 Web 服务器 (contweb)

概述：

恢复先前暂停的指定 Web 服务器。

用法：

contweb <--ADSPath|-a server1[,server2,server3...]>
[--computer|-c COMPUTER1[,COMPUTER2...]]
[--verbose|-v]
[--help|-?]

参数：

-a SERVERx 要恢复的服务器的编号。
-Computer 逗号分隔的一组计算机。例如 -c Machine1,Machine2,Machine3。
这里假设指定路径在所有指定的计算机上都存在。如果未指定计算机，则认为是本机。

示例：

contweb -a 1,2

contweb -c MACHINE1,MACHINE2,MACHINE3 -a 1,2

关于管理脚本

本主题包含了有关管理脚本（包括 IIS Admin 对象、Cscript.exe 命令行脚本实用程序以及管理脚本的默认安装位置）的概念信息。

  • IIS Admin 对象
  • Cscript.exe

IIS Admin 对象

管理脚本将一些最通用的管理任务进行了自动化处理并演示了 IIS Admin 对象的功能。您可以使用这些对象创建和控制网站、应用程序、目录等。

可以使用默认安装到系统驱动器上 \InetPub\adminscripts 目录中的 VBScript 文件运行管理脚本。

Cscript.exe

如果将 Cscript 注册为可以运行 .vbs 文件，VBScript 管理脚本即可被 Cscript.exe 命令行脚本实用程序使用并正常运行。

要注册 Cscript，请运行 adsutil.vbs 示例。它将帮助您注册 Cscript.exe。

也可以通过在命令行提示符运行下列命令，将 Cscript.exe 注册为默认脚本主机：

cscript //H:cscript

注册 Cscript 后，可以直接调用脚本而不必指定脚本引擎。例如，可以在命令行提示符键入以下命令：

pauseweb.vbs /?

替代：

cscript pauseweb.vbs /?

然而，某些脚本会将输出发送到弹出窗口而不是命令行窗口，这在输出内容很多时就会很麻烦。为避免出现弹出窗口，应保持在脚本名称前面使用 cscript。

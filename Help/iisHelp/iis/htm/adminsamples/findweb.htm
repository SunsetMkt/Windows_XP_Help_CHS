查找网站 (findweb)

概述：

查找指定计算机上的命名网站。

用法：

findweb [--computer|-c COMPUTER] WEBSITE [--help|-?]

参数：
Computer 要搜索的计算机。如果未指定，将搜索本机。
WEBSITE 要搜索的网站。如果未指定，将搜索“1”。
此名称可以将此参数指定为如下名称之一，按指定的优先级：
服务器编号（例如，1、2、10 等）
服务器说明（“My Server”）
服务器主机名称 (www.domain.com)
IP 地址 (172.16.0.0)
help 返回该命令的帮助。

示例：

findweb 

findweb -c MACHINE www.domain.com

Active Server Pages 指南

欢迎使用 Active Server Pages 指南。本节提供了有关如何创建 ASP 网页及开发 Web 应用程序的详细信息。还将说明可用于创建组件的特定事件方法和接口，以供 ASP 或 ISAPI 扩展访问使用。在本节中，您将了解到有关如何存储自定义的 IIS 配置数据以及如何使用内置对象来处理这些数据的信息。最后，本指南提供了一个脚本和程序示例库，用于演示与 IIS 进行交互的各种方法。

本节包含：

  • Active Server Pages：介绍如何创建 Active Server Pages，并阐述许多与如何理解 IIS 编程平台有关的关键概念。
  • 开发 Web 应用程序：概述有关 IIS 如何满足 Microsoft 分布式应用程序体系结构要求的概念，同时提供如何设计和实施应用程序以便通过 HTTP 进行部署的特定详细信息。
  • 以编程方式管理 IIS：提供帮助您以程序方式管理 IIS 配置的概述和参考资料。
  • 内置 ASP 对象：描述 ASP 内置对象的属性和方法。
  • ASP 的可安装组件：概述可安装用于与 ASP 一起使用的各种组件。
  • 脚本参考：包含各种参考资料，包括 @ 指令和 Global.asa 参考。
  • ASP 示例：把您带到 Platform SDK 的 IIS 一节或 MSDN Online - Internet Information Services 中的 ASP 脚本示例。

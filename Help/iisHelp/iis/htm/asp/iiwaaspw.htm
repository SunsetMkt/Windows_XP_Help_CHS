了解 Active Server Pages

Active Server Pages (ASP) 使生成动态 Web 内容和建立功能强大的 Web 应用程序变得十分简单。无论您是 Web 设计人员还是 Web 开发人员，此简介中关于 ASP 的说明都会对您大有裨益。

本节包含：

  • Active Server Pages 简介：包含 Active Server Pages 的概述信息。
  • ASP 新增功能：介绍此版本中新添加的功能。
  • ASP 的重要变化：介绍此版本中有所变化的功能及其用法。
  • ASP 教程：有关如何使用 ASP 创建服务器端脚本的循序渐进指导课程。

本节不包含：

  • 有关如何使用 ASP 编写服务器端脚本的基本概念；请参阅建立 ASP 网页。
  • 有关编写更高级脚本的信息；请参阅 Web 应用程序：概述。

HTTP 错误 414 - 请求 - URI 太长
Internet 信息服务

技术信息（用于支持人员）

  • 背景

    请求的 URI 太长，服务器拒绝处理请求。这种情况仅在下列情形下才会发生：

    客户端错误地将 POST 请求转换为带有长查询信息的 GET 请求。

    客户端遇到重定向问题（例如，重定向 URL 的前缀指向自身的后缀）。

    服务器遭到客户端的攻击，该客户端试图找出那些使用定长缓冲区来读取或控制请求 URI 的服务器上的安全漏洞。
